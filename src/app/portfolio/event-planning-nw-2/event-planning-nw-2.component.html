Event Planning NW (redesign)

This is an event planning site for a business in Happy Valley. It was built with MEAN stack using Bootstrap 4. It includes APIs for authorized CRUD operations, authentication and access management with automatic JWT renewal (via OAuth), and simple/complex forms with custom validation. Angular routing is used for navigation. There was more effort in using various component styles in this project. Sliders, carousels, animations and more attention to responsive development for mobile using Bootstrap grid was employed. This site is still a work in progress.
